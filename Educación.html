Aprendizaje y Crecimiento

Como ingeniera de software, creo en el aprendizaje continuo y en mantenerme al día con las últimas tecnologías y mejores prácticas. A continuación, se presentan algunas áreas clave en las que actualmente estoy profundizando:

  • Programación Orientada a Objetos (OOP): Fortaleciendo mi comprensión de los principios de OOP para escribir código más limpio, escalable y reutilizable
  • Principios SOLID: Aprendiendo y aplicando los principios SOLID para mejorar el diseño del software y crear aplicaciones más limpias y robustas
  • Amazon Web Services (AWS): Adquiriendo experiencia práctica con AWS para desplegar, gestionar y escalar aplicaciones en la nube
  • Arquitectura de Microservicios: Explorando el diseño e implementación de microservicios para crear sistemas altamente escalables y tolerantes a fallos
  • Desarrollo Guiado por Pruebas (TDD): Adoptando prácticas TDD para escribir mejores pruebas y mejorar la fiabilidad de mis aplicaciones
  • Repositorios JPA: Aprendiendo a trabajar con repositorios JPA (Java Persistence API) para interactuar eficientemente con bases de datos y aprovechando Spring Data JPA para simplificar el acceso a los datos
  • Arquitectura Hexagonal: Entendiendo la Arquitectura Hexagonal (Puertos y Adaptadores) para diseñar sistemas mantenibles, probables y desacoplados
