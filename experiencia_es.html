Aplicación de Guardado de Notas (Configuración Backend)

• Desarrollé una aplicación de guardado de notas utilizando Spring Boot con Repositorios JPA para interactuar con una base de datos en memoria H2, simplificando las operaciones CRUD y permitiendo un desarrollo local rápido.

• Utilicé Lombok para reducir el código repetitivo generando automáticamente los métodos getters, setters, y constructores mejorando el mantenimiento del código.

• Apliqué Arquitectura Hexagonal para estructurar la aplicación con una clara separación entre la lógica central, el acceso a datos y las dependencias externas, lo que la hace más mantenible y fácil de probar.

• Configuré la consola H2 de Spring Boot para realizar pruebas y depuración locales de manera eficiente, mientras preparaba el sistema para su eventual despliegue a una base de datos de producción.
